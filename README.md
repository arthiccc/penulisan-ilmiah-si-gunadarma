# Proyek Penulisan Ilmiah (PI) - Sistem Informasi

Selamat datang di repository panduan Penulisan Ilmiah (PI) untuk mahasiswa S1 Sistem Informasi. Repository ini telah dilengkapi dengan **MCP (Model Context Protocol) Skill: Dosen Pembimbing** untuk membantu Anda dalam proses penulisan.

## Isi Repository

- `docs/`: Dokumen resmi Pedoman PI 2025, Prosedur Sidang, dan materi workshop.
- `skills/dosen-pembimbing/`: Agent skill yang dapat digunakan oleh AI (seperti Gemini/Claude) untuk bertindak sebagai pembimbing virtual Anda.
- `templates/`: Template struktur penulisan per bab.

## Cara Menggunakan Agent Dosen Pembimbing

Jika Anda menggunakan AI yang mendukung MCP Skills (seperti Gemini CLI atau Claude Desktop):
1. Pastikan folder `skills/` terdaftar dalam konfigurasi AI Anda.
2. Anda bisa langsung bertanya:
   - *"Tolong cek apakah margin draf saya sudah sesuai pedoman PI 2025?"*
   - *"Bantu saya membuat latar belakang untuk topik Implementasi AI di E-commerce."*
   - *"Apa saja bentuk PI yang diperbolehkan tahun ini?"*

## Panduan Cepat (Quick Start)

1. **Pilih Topik:** Lihat di `docs/Sistem Informasi/` untuk inspirasi topik dari workshop terbaru (AI, Big Data, BI, Dev, dsb).
2. **Gunakan Template:** Mulai menulis menggunakan panduan di `templates/README.md`.
3. **Cek Format:** Gunakan Skill Dosen Pembimbing untuk memvalidasi format teknis Anda sebelum bimbingan ke dosen asli.

## Persyaratan Teknis Utama (Checklist)
- [ ] Kertas A4 80gr.
- [ ] Font Times New Roman 12.
- [ ] Spasi 1.5.
- [ ] Margin: 4-3-4-3 (Atas-Bawah-Kiri-Kanan).
- [ ] Minimal 10 Pertemuan Bimbingan untuk ACC Sidang.
